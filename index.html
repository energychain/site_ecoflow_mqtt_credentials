MQTT Credential Extraction for Ecoflow (API)

Ecoflow Powerstations might be controlled via the Ecoflow APP. However, using the APP it is not possible to use the data in Smart-Home appliances or Energy Management. This page uses the login credentials to retrieve login credentials for MQTT access.

Security Note

The Javascript used on this page communicates only with the servers of Ecoflow. No data is stored or transfered to another party (like us). Please check the source of the page and the script to validate!
